Project description

"A Web Application to enables users to provide online tests, review them, and display the results."

It consists of One section:
Admin: Admin can view, add, update and delete questions of test
User: User can give the test and see the test result

Features:
• Admin Module
• Test Module
• Display Result
• Update questions details
• Delete questions details

Tech Stack

Visual Studio Code
Frontend: Angular
Database: MYSQL
